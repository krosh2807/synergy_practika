# Кейс-задача №1. Стилистическое преобразование чисел

Консольная программа на Python.

## Функциональность
1. Запрашивает у пользователя день, месяц и год рождения.
2. Определяет день недели введённой даты.
3. Определяет, был ли год рождения високосным.
4. Вычисляет текущий полный возраст пользователя.
5. Выводит дату рождения в формате дд.мм.гггг, где каждая цифра отрисована
   звёздочками (*), как на электронном табло.

## Запуск
```bash
python3 birthday.py
```

## Пример работы
```
Введите день рождения (1-31): 15
Введите месяц рождения (1-12): 6
Введите год рождения: 2000

Вы родились в четверг.
Год 2000 високосный.
Сейчас вам 26 лет.

Ваша дата рождения на электронном табло:
...
```
