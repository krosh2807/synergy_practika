# Кейс-задача №2. Счётчик (Counter) со стилизацией интерфейса

HTML + CSS + Vanilla JS.

## Функциональность
- Кнопки "+" и "−", окно отображения результата.
- Фон страницы — линейный градиент, все элементы отцентрованы.
- Кнопки со скруглением 5px и градиентным фоном.
- Клик по "+" увеличивает значение на 1, по "−" — уменьшает на 1.
- Цвет окна результата: жёлтый (значение > 0), зелёный (< 0), красный (= 0).
- При достижении 10 кнопка "+" блокируется, при −10 блокируется кнопка "−".
- При достижении 10 или −10 под кнопками появляется сообщение
  "вы достигли экстремального значения".

## Файлы
- `index.html` — разметка
- `style.css` — стили
- `script.js` — логика

## Запуск
Открыть `index.html` в браузере.
