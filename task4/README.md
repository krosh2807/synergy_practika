# Кейс-задача №4. Интерактивный калькулятор

HTML + CSS + Vanilla JS.

## Функциональность
- Интерфейс из двух полей ввода и четырёх кнопок: "сумма", "разность",
  "произведение", "деление".
- Пустое окно вывода результата.
- Каждая кнопка вызывает соответствующую математическую функцию
  (сложение, вычитание, умножение, деление) над введёнными числами.
- При вводе нечисловых значений в окне результата выводится сообщение
  об ошибке. Деление на ноль также обрабатывается отдельным сообщением.

## Файлы
- `index.html` — разметка
- `style.css` — стили
- `script.js` — логика

## Запуск
Открыть `index.html` в браузере.
